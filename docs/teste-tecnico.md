# Teste Técnico

## Requisitos funcionais

1. Controlar abastecimentos diários registrando bomba utilizada, quantidade de litros e valor abastecido.
2. Registrar imposto de 13% incidente sobre cada abastecimento.
3. Mapear bombas a tanques: dois tanques (gasolina e óleo diesel), cada um ligado a duas bombas.
4. Gerar relatório agrupando abastecimentos (dia, tanque, bomba, valor) e total geral do período.

## Requisitos não funcionais

1. Aplicar boas práticas e demais conhecimentos relevantes.
2. Criar testes unitários/funcionais conforme necessidade.
3. Utilizar somente componentes/recursos nativos da versão escolhida do Delphi.
4. Construir o relatório com Fortes Report.
5. Utilizar banco Firebird, entregando script de criação e arquivo do banco.

## Requisitos não funcionais com pontuação adicional

1. Tempo de entrega.
2. Uso do GitHub para publicação do projeto.
3. Orientação a Objetos.
4. Firedac.
